# 中文引号支持 - 快速开始指南

## 什么是中文引号？

中文引号是用于标记字符串的**弯引号**，与英文的直引号不同：

```
英文引号：" "  ' '  ` `
中文引号："  "  '  '（弯引号）
```

## 在 ZhCode 中使用

### ✅ 现在可以这样写：

```zhcode
// 使用中文双引号
令 message = "你好，世界"
打印(message)

// 使用中文单引号  
令 user = '用户名'
打印('欢迎，' + user)

// 在函数中
函数 greet(name) {
  返回 "欢迎，" + name
}
```

### 编译结果（JavaScript）：

```javascript
let message = "你好，世界";
打印(message);

let user = "用户名";
打印('欢迎，' + user);

function greet(name) {
  return "欢迎，" + name;
}
```

## 引号配对规则

### 中文双引号 " "

```
打开：U+201C（左双引号）
关闭：U+201D（右双引号）

示例：
"这是一个完整的字符串"
```

### 中文单引号 ' '

```
打开：U+2018（左单引号）
关闭：U+2019（右单引号）

示例：
'这是另一个字符串'
```

## ⚠️ 注意事项

1. **引号必须配对** - 左引号必须有对应的右引号
   ```zhcode
   ✅ 正确：令 x = "内容"
   ❌ 错误：令 x = "内容'
   ```

2. **混合类型也可以** - 可以在同一行使用不同引号
   ```zhcode
   ✅ 正确：令 a = "一个" + '另一个'
   ```

3. **转义字符** - 在字符串内使用转义序列
   ```zhcode
   令 s = "双引号：\" 和单引号'"
   令 t = "换行：\n新行"
   ```

## 从哪里获取中文引号？

### 在 Windows 上：

1. **输入法**（中文输入法）
   - 在中文输入法中，按 Shift 可能有引号快捷键
   
2. **特殊字符插入**
   - 在 VS Code 中使用 Ctrl+Shift+U 输入 Unicode
   - 例如：201c（Enter）= "左双引号

3. **复制粘贴**
   - 从本文档或示例代码中复制

4. **Character Map**（字符映射表）
   - Windows 系统工具 → 搜索"charmap"
   - 搜索"U+201C"找到左双引号

## 常见问题 (FAQ)

### Q: 为什么看到的引号是弯的而不是直的？

A: 这些是特殊的 Unicode 字符：
- U+201C/201D = 智能（弯）引号
- U+0022/0027 = 直引号（键盘上的）

ZhCode 同时支持两种。

### Q: 可以混合使用英文和中文引号吗？

A: **可以**
```zhcode
令 a = "中文"
令 b = "英文"
令 c = '混合'
```
编译器会正确处理任何组合。

### Q: 为什么我的引号不起作用？

A: 检查以下几点：
1. 左右引号是否配对（" 和 "，' 和 '）
2. 是否复制了错误的引号类型
3. 运行 `npm run test` 检查 Tokenizer 是否识别

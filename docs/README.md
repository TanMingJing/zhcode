## 🔑 关键概念快速查询

### 撤销/重做
- **文件**: App.tsx
- **快捷键**: Ctrl+Z / Ctrl+Y
- **详情**: 每个文件独立的 Undo/Redo 栈
- **文档**: QUICKSTART.md, ARCHITECTURE.md

### 云端项目
- **后端**: Appwrite
- **集合**: zhcode_projects
- **按钮**: ☁️
- **文档**: CLOUD_PROJECTS.md, ARCHITECTURE.md

### 导出功能
- **格式**: JavaScript (.js) + JSON (.zhcode.json)
- **按钮**: ⬇️
- **编译器**: @zhcode/core
- **文档**: CLOUD_PROJECTS.md, FEATURES_UPDATE.md

### AI 历史
- **后端**: Appwrite
- **集合**: ai_operations
- **按钮**: 📖
- **记录数**: 50 条
- **文档**: CLOUD_PROJECTS.md, FEATURES_UPDATE.md

## 🔗 快速链接

### 重要文件位置
- **主组件**: `packages/ide/src/App.tsx`
- **服务**: `packages/ide/src/services/appwriteService.ts`
- **配置**: `.env` 文件

### Appwrite 配置
```
VITE_APPWRITE_PROJECT_ID=6940e8610022e30d684a
VITE_APPWRITE_DATABASE_ID=zhcode_db
```

### 启动命令
```bash
cd packages/ide
npm install
npm run dev
# 访问 http://localhost:3001
```

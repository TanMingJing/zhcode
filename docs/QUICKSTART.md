# 🚀 ZhCode IDE - 快速开始指南

## 📥 安装和启动

### 1. 开始开发服务器

```bash
# 进入 IDE 目录
cd packages/ide

# 安装依赖（如果还没装过）
npm install

# 启动开发服务器
npm run dev
```

### 2. 打开浏览器

访问：**http://localhost:3001**

---

## ✏️ 第一个 ZhCode 程序

### 编写代码

在编辑器左侧创建一个新文件 `hello.zhc`，写入：

```zhcode
函数 加法(a, b) {
  返回 a + b
}

变量 结果 = 加法(5, 3)
打印(结果)  // 输出：8
```

### 运行代码

点击 **▶️ 运行** 按钮，在右侧输出区查看结果：

```
8
```

## 🎯 核心功能演示

### 1. 撤销/重做

```
编辑代码 → Ctrl+Z（撤销）→ Ctrl+Y（重做）
```

**快捷键**：
- `Ctrl+Z`：撤销上一步
- `Ctrl+Y`：重做上一步
- `🕐` 按钮：重置文件到原始状态

---

### 2. 保存项目到云端

#### 步骤

1. 编写完代码后，点击 **☁️ Cloud Projects** 按钮
2. 输入项目名称，例如："我的计算器"
3. 输入项目描述（可选）
4. 点击 **"Save to Cloud"** 按钮
5. ✅ 项目已保存到 Appwrite

#### 查看已保存的项目

1. 再次点击 **☁️** 按钮
2. 在 **Projects** 列表中看到你的项目
3. 显示信息：
   - 项目名称
   - 项目描述
   - 文件数量
   - 最后修改时间

---

### 3. 从云端加载项目

#### 步骤

1. 点击 **☁️ Cloud Projects** 按钮
2. 在项目列表中找到要加载的项目
3. 点击项目右侧的 **Load** 按钮
4. ✅ 项目已加载到编辑器

#### 继续编辑

加载后可以继续编辑，再次保存会更新云端项目

---

### 4. 删除云端项目

#### 步骤

1. 点击 **☁️ Cloud Projects** 按钮
2. 找到要删除的项目
3. 点击项目右侧的 **Delete** 按钮
4. 确认删除
5. ✅ 项目已从云端删除

---

### 5. 导出为 JavaScript

#### 单个文件导出

1. 在编辑器编写 ZhCode
2. 点击 **⬇️ Export Code** 按钮
3. 自动转译为 JavaScript
4. 文件自动下载为 `.js` 格式

#### 例子

**ZhCode 代码**：
```zhcode
函数 问候(名字) {
  返回 `你好，${名字}！`
}
```

**导出的 JavaScript**：
```javascript
function 问候(名字) {
  return `你好，${名字}！`;
}
```

#### 导出整个项目

项目保存到云端时会自动为 `project.zhcode.json` 格式。
也可以选择导出为单个 `.js` 文件进行集成。

---

### 6. 查看 AI 操作历史

#### 打开历史面板

1. 点击 **📖 AI History** 按钮
2. 显示最近 50 条 AI 操作记录

#### 历史记录包含

- **操作类型**：Generate（生成）/ Explain（解释）/ Optimize（优化）
- **输入内容**：你给 AI 的指令
- **输出内容**：AI 生成的代码
- **编程语言**：检测为中文/英文
- **框架类型**：检测为 ZhCode/React/JavaScript
- **操作状态**：✓ 成功 / ✗ 失败
- **时间戳**：操作的准确时间

#### 使用场景

- 回溯 AI 改动历史
- 学习 AI 优化思路
- 分析 AI 错误原因
- 追踪项目变化

## 🤖 AI 功能使用

### 生成代码

1. 在输入框输入需求，例如："生成一个斐波那契数列函数"
2. 点击 **🧠 Generate** 按钮
3. AI 自动生成代码
4. 代码自动添加到编辑器
5. 操作自动记录到历史

### 解释错误

1. 编写代码但出现错误
2. 运行代码看到错误信息
3. 点击 **💡 Explain** 按钮
4. AI 解释错误原因
5. 点击代码片段应用建议修复

### 优化代码

1. 编写完整代码
2. 点击 **✨ Optimize** 按钮
3. AI 分析并提出优化建议
4. 一键应用优化
5. 查看改进前后对比

## 🌐 多语言支持

### 切换语言

1. 打开 **Settings** 面板
2. 找到 **Language** 选项
3. 选择 **中文** 或 **English**

### 效果

- 所有 AI 生成的代码都按选定语言输出
- 错误解释按选定语言展示
- 代码建议按选定语言生成

**例子**：
- 中文模式：生成中文关键字代码（函数、返回等）
- 英文模式：生成英文关键字代码（function、return等）

## ⚠️ 常见问题

### Q: 我的代码可以保存到云端吗？

**A**: 是的！使用 ☁️ Cloud Projects 按钮保存。

### Q: 导出的 JavaScript 可以在浏览器运行吗？

**A**: 可以！导出的 JavaScript 是标准 ECMAScript，支持浏览器或 Node.js 运行。

### Q: AI 生成的代码准确性如何？

**A**: AI 使用 GPT 模型生成，准确率约 85-95%。复杂逻辑建议人工审查。

### Q: 我的项目数据安全吗？

**A**: 所有数据加密存储在 Appwrite，只有你能访问。支持自托管 Appwrite。

### Q: 能导出为 React 组件吗？

**A**: 可以！写 JSX 语法的 ZhCode，导出后得到 React.createElement() 调用。

### Q: 支持团队协作吗？

**A**: 当前版本不支持实时协作。可以通过分享 .zhcode.json 文件进行版本交换。
